Frontend simple para el proyecto SIG

Archivos:
- index.html: entrada principal (abrir en el navegador)
- styles.css: estilos
- app.js: lógica de interacción (formulario, lista, tema)

Cómo probar:
1. Abrir `frontend/index.html` en tu navegador (doble clic o arrastrar al navegador).
2. Probar el formulario y añadir elementos a la lista.

Notas:
- El toggle de tema usa `localStorage` para recordar la preferencia.
- Si quieres integrar con el backend, puedo añadir llamadas `fetch()` para consumir la API.
