# manhwav7

Bienvenido a **Mangaverse**, tu espacio para explorar, compartir y disfrutar del mundo del manga.

## Características

- Explora una amplia colección de mangas.
- Guarda tus favoritos.
- Comparte recomendaciones con la comunidad.

## Instalación

```bash
git clone https://github.com/joker627/manhwav7.git
cd manhwav7
npm install
```

## Uso

```bash
npm start
```

Abre tu navegador en `http://localhost:3000`.
